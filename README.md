# School Management Database System 🎓

Цей проєкт реалізує реляційну базу даних для управління інформацією про студентів, викладачів, курси та оцінки за допомогою **SQLAlchemy** та **Alembic**.

## 🛠 Технологічний стек
- **Python 3.10+**
- **SQLAlchemy 2.0+** (ORM для роботи з БД)
- **Alembic** (управління міграціями)
- **PostgreSQL** (основна база даних)
- **Faker** (генерація тестових даних)

## 📋 Структура бази даних
- **Students**: список студентів та їх приналежність до груп.
- **Groups**: академічні групи.
- **Teachers**: викладачі.
- **Subjects**: перелік предметів із закріпленими за ними викладачами.
- **Grades**: оцінки студентів з предметами та датою отримання.

## 🚀 Швидкий старт

### 1. Підготовка середовища
Клонуйте репозиторій та встановіть залежності:
```bash
pip install -r requirements.txt
```

### 2. Запуск PostgreSQL
Запустіть контейнер Docker (приклад налаштувань):
```bash
docker run --name school-postgres -p 5432:5432 -e POSTGRES_PASSWORD=mysecretpassword -d postgres
```
*Налаштуйте підключення у файлі `.env`.*

### 3. Міграції та Наповнення
Створіть структуру таблиць та заповніть їх випадковими даними:
```bash
python -m alembic upgrade head
python seed.py
```

## 🔍 Аналітичні запити (`my_select.py`)
Проєкт містить 10+ функцій для аналізу даних:
- `select_1`: 5 студентів із найбільшим середнім балом.
- `select_2`: Найкращий студент з певного предмета.
- `select_5`: Курси, які читає певний викладач.
- *... та багато інших.*

Для перевірки виконайте:
```bash
python my_select.py
```

## 🛠 CLI Програма (`main.py`)
Для зручного управління даними реалізовано CLI-інтерфейс (CRUD операції):

**Створити викладача:**
```bash
python main.py -a create -m Teacher -n "Boris Jonson"
```

**Показати всі групи:**
```bash
python main.py -a list -m Group
```

**Оновити дані студента:**
```bash
python main.py -a update -m Student --id 1 --name "New Name"
```
